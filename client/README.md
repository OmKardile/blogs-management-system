# Blogs Management System

This is a simple blogs management system built with React, Node.js, Express and MongoDB.

## Features

- User authentication
- User can create, read, update and delete blogs
- User can view all blogs
- User can view a single blog

## How to use

- Clone the repository
- install dependencies for both client and server
- start the server
- start the client
- use the application

## Technologies Used

- React
- Node.js
- Express
- MongoDB
